## Highlights

- Initial stable release on the new Tauri runtime.
- Full release history view inside the About dialog with direct links to each GitHub release.
- After a successful in-app update and relaunch, the changelog modal opens automatically focused on the version that was just installed.
- Hardened local browser-preview fallbacks so desktop bridge verification flows stay stable.

## Fixes

- Fixed Windows in-app update: pressing "Update" now reliably installs the new version. The previous code path closed the running app but never actually installed the downloaded NSIS package, so users had to restart manually only to find the old version still in place.

## Verification

- macOS, Linux, and Windows production bundles built from CI.
- Updater manifest signed and published from the release workflow.
